# 项目简介

## 核心功能
- 基于 YOLOv8 的夜间车辆检测系统，支持 8 类目标检测：car、pedestrian、traffic light、traffic sign、bicycle、bus、truck、rider
- 提供完整的源码、5000 张标注数据集、训练好的模型及训练结果
- 支持多种检测方式：单张图片、图片文件夹、视频文件、本地摄像头
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 支持自定义置信度和 IOU 阈值
- 支持选择显示标签和原图
- 支持选择检测模型

## 适用场景
- 夜间车辆检测与分析
- 交通监控系统
- 智能安防应用
- 行人检测与交通标志识别
- 批量图像与视频处理

## 技术特点
- 基于 Python 3.9，使用 OpenCV、PyQt5、Ultralytics、PyTorch 1.9 等技术栈
- 提供 PyCharm+Anaconda 或 VSCode+Anaconda 开发环境
- 包含完整 UI 界面源文件和图标
- 配置成功环境后可直接使用
- 支持实时预览和批量处理功能
